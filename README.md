## Social Big Data-CM

### Ingest_socialbd

This is a project master degree about the analysis of social change from Big Data.

Full documentation is available in docs/Memoria_JeffersonAlmacheMontoya.pdf (Spanish)

### Used technologies: 
- Scala 2.11
- Apache Flink v1.3.0
- Elastic Search 5.2
- Kibana 5.2
- Apache Zeppelin 0.7
- Kafka 0.10.1.0
- Maven
